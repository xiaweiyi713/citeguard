## Release Summary

CiteGuard is now public as an early-stage falsification-first research agent prototype. This release focuses on making the project runnable, inspectable, and extensible for collaborators working on trustworthy scientific writing and citation verification.

## Highlights

- End-to-end `claim -> citation -> evidence` pipeline
- Claim-level verification with existence, metadata, support, and contradiction checks
- Multi-source scholarly adapters for OpenAlex, Crossref, arXiv, and Semantic Scholar
- Real reranker + NLI support verification path with heuristic fallback
- Best-effort live evidence chunk harvesting from scholarly landing pages
- Systematic support-threshold and ensemble-weight calibration tooling
- Audit-friendly outputs with evidence provenance
- Public repository essentials: install path, CI, examples, license, citation metadata, and contribution docs

## Included in This Release

### Core system

- `CCEG` graph model
- planning, retrieval, citation proposal, verification, risk fusion, and constrained writing
- audit report generation

### Verification and evidence

- heuristic support baseline
- sentence-transformer or cross-encoder reranker support backend
- transformers NLI backend
- calibratable ensemble policy
- live evidence chunk ingestion into `SupportVerifier`

### Research tooling

- demo evaluation script
- support calibration script
- example corpus and example calibration set
- unit tests covering retrieval, verifiers, writing, and calibration

## Known Limitations

- This is a research prototype, not a production citation service
- Benchmark coverage is still small and should be expanded with human-reviewed labels
- Live evidence harvesting is best-effort and source dependent
- Some publisher landing pages expose no usable text
- The API layer is still minimal

## Recommended First Commands

```bash
python3 -m venv .venv
. .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e .
python3 -m unittest discover -s tests -v
python3 scripts/run_agent.py --topic "citation hallucination in scientific writing" --corpus examples/demo_corpus.json --section-count 1 --support-mode heuristic
```
