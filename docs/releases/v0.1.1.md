# CiteGuard v0.1.1

`v0.1.1` is the first release published to PyPI under the distribution name
`citationguard`, and the first release with structured Chinese-bibliography
support.

## Release Summary

CiteGuard v0.1.1 makes Chinese academic writing a first-class input while
keeping the conservative falsification-first stance:

- **GB/T 7714 reference parsing.** Chinese national-standard references
  (`作者. 标题[J]. 期刊, 年, 卷(期): 页`, plus `[M]/[C]/[D]/[R]` and `/OL`
  carrier variants, in Chinese and English) are parsed into structured
  title/authors/venue fields in free-text parsing, file extraction, and batch
  audits.
- **Registrar-agnostic DOI existence checks.** When open sources return
  `not_found` for a citation that carries a DOI, CiteGuard now consults the
  global doi.org Handle registry (which covers China DOI/ISTIC and every other
  registrar) and reports `doi_registration` with the official resolution URL.
  A registered DOI means "metadata not open", never "paper does not exist";
  an unregistered DOI lowers confidence further but is still not fabrication
  proof. Disable with `CITEGUARD_DOI_REGISTRY=0`.
- **CJK query routing.** Crossref bibliographic search cannot match
  predominantly-CJK queries, so those wasted calls are skipped; DOI lookups
  are unaffected.
- **PyPI distribution rename.** The package installs as `citationguard`
  (`pip install citationguard`) because the `citeguard` name on PyPI belongs
  to an unrelated project; the import package stays `citeguard` and the
  `citeguard` / `citeguard-mcp` console commands are unchanged.
- **Bilingual documentation.** `README.md` is now Chinese-first with a full
  English companion in `README.en.md`, both checked by release gates.
- **Toolchain.** Ruff linting and incremental mypy type checking run in CI;
  the legacy root-package shims and `setup.py` were removed for a
  pyproject-only build.

The support eval remains a synthetic seed fixture, not a human-reviewed
benchmark; release wording keeps that boundary explicit.

## Recommended First Commands

For an installed or published package:

```bash
python -m pip install citationguard
python -m pip install "citationguard[mcp]"   # MCP server, requires Python >= 3.10
citeguard status
citeguard verify --title "Attention Is All You Need" --arxiv-id 1706.03762 --year 2017
citeguard audit refs.md            # Markdown/LaTeX/BibTeX/BBL/DOCX + GB/T 7714
```

For source checkout release rehearsal:

```bash
python -m pip install -e .
python -m pip install -e ".[mcp]"
python3 -m unittest discover -s tests -v
```
